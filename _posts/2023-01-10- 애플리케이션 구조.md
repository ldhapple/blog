여러가지 애플리케이션 구조가 있지만 가장 단순하면서 많이 사용하는 방식은 역할에 따라 위 3가지 계층으로 나누는 것이다.

- 프레젠테이션 계층
	- UI와 관련된 처리 담당
	- 웹 요청과 응답
	- 사용자 요청을 검증
	- 주 사용 기술 : 서블릿과 같은 HTTP 웹 기술, 스프링 MVC
- 서비스 계층
	- 비즈니스 로직을 담당
	- 주 사용 기술 : 가급적 특정 기술에 의존하지 않고, 순수 자바 코드로 작성.
- 데이터 접근 계층
	- 실제 데이터베이스에 접근하는 코드
	- 주 사용 기술 : JDBC, JPA, File, Redis, Mongo ...
